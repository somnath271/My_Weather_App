How It Works

  1. Search a City: Start typing your city in the search bar. You’ll see smart suggestions just below the input.
  2. Secure API: We request current conditions from our weather API through serverless functions that keep your API key safe.
  3. Instant Results: The app shows temperature, humidity, wind and sunrise/sunset. Common cities update in the background.
  4. Mobile Friendly: Everything is optimized for phones and tablets for on‑the‑go checks.
